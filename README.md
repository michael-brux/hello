# hello
Hello World Scripts

## run bash script
To print "Hello World!" by running the [world.bash](https://github.com/michael-brux/hello/blob/main/world.bash) script:
```bash
curl -sL https://raw.githubusercontent.com/michael-brux/hello/refs/heads/main/world.bash | bash
```
## run Python script
To run the "Hello World!" Python script:
```bash
curl -sL https://raw.githubusercontent.com/michael-brux/hello/refs/heads/main/world.py | python3
```
